Краткое руководство по установке

  1. Подготовка окружения: Убедитесь, что ваш сервер настроен на Debian 12, с установленными Nginx, Python и Gunicorn.
  2. Загрузка и установка: После скачивания распакуйте архив и установите зависимости с помощью команды: 
    pip install -r requirements.txt
  3. Настройка: Задайте необходимые ключи API в .env файле, согласно документации.
  4. Запуск: Запустите приложение через Gunicorn: 
    gunicorn -w 4 -b 127.0.0.1:8000 wsgi:app
    Не забудьте настроить проксирование на Nginx.
