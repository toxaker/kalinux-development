Краткое руководство по установке

  1. Подготовка окружения: Убедитесь, что ваш сервер настроен на Debian 12 с установленными:
    • Web-сервер: Nginx
    • Язык программирования: Python 3.10+
    • WSGI-сервер: Gunicorn
  2. Загрузка и установка: Распакуйте скачанный архив и выполните установку зависимостей: 
    pip install -r requirements.txt
  3. Настройка: Создайте файл .env и заполните его ключами API и другими параметрами из документации.
  4. Запуск: Выполните запуск через Gunicorn: 
    gunicorn -w 4 -b 127.0.0.1:8000 wsgi:app
    Затем настройте проксирование на уровне Nginx.

Для более детальных инструкций ознакомьтесь с Документацией.
